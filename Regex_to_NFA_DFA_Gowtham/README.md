
# Regex to NFA and DFA Converter

This project implements a complete pipeline to convert a **Regular Expression (Regex)** into:
- **Nondeterministic Finite Automaton (NFA)**
- **Deterministic Finite Automaton (DFA)**
- **Minimized DFA**

The implementation is done in **Python** and demonstrates core concepts of **Automata Theory**.
